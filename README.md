# Pricing Page

This is a static pricing page website designed to be deployed on Vercel.

## Structure
- `index.html`: Main structure.
- `styles.css`: Styling (Vanilla CSS).
- `script.js`: Interactive logic (Toggle).
- `assets/`: Logos.

## Deployment
1. Initialize a git repository: `git init`
2. Commit files: `git add . && git commit -m "Initial commit"`
3. Push to GitHub.
4. Import project in Vercel. It will automatically detect the static site.

## Notes
- Logos are in vector format (SVG) where possible.
- The design is responsive and uses a premium aesthetic with 'Playfair Display' and 'Inter' fonts.
